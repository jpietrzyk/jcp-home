# Web App (Vite + React)

## Quick start

1. Install dependencies:

```bash
npm install
```

2. Copy environment variables:

```bash
cp .env.example .env
```

3. Run local dev server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

## Notes

- Routing is defined in `src/router.tsx`.
- CMS integration stubs are in `src/lib/cms`.
- Replace fallback content by setting valid Sanity env vars.
